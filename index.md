## Welcome to Mega css

Megacss the future of tommorow. The open source framework for css.

### Get Started

Use Cdn to use megacss on website

``` html
<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/megacss/mega.css@main/mega.css">
```
